A **Swiggy-inspired food delivery UI** built using React and enhanced with a **real-world DevOps pipeline**.

✔️ Modern UI Design
✔️ Fully Responsive Layout
✔️ DevOps Integration (CI/CD + Docker)
✔️ Production-ready structure

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Vishalkondi/Swiggy-application.git
cd Swiggy-application
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Run Application

```bash
npm start
```

👉 Runs on: http://localhost:3000

---

## 📦 Build for Production

```bash
npm run build
```

---

## 🌍 Deployment

### GitHub Pages

```bash
npm run deploy
```

---

### Docker

```bash
docker build -t swiggy-app .
docker run -p 3000:3000 swiggy-app
```
